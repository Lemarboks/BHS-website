# Bloubergrant High School React + TypeScript Website

This is a modern animated React/Vite redesign concept for Bloubergrant High School.

## Files included

- `index.html` - Vite entry HTML file
- `src/main.tsx` - React entry file
- `src/App.tsx` - Main app and routes
- `src/styles.css` - Full website styling
- `src/data.ts` - School content, navigation, cards, staff, subjects and news
- `src/components/` - Header, Footer, Hero, animated reveal wrapper and reusable UI sections
- `src/pages/` - Separate React pages: Home, About, Curriculum, Admissions, Resources, Events, News, Staff and Contact
- `src/assets/images/` - Local SVG logo and image assets
- `package.json` - Project dependencies and scripts
- `tsconfig.json` - TypeScript configuration
- `vite.config.ts` - Vite/React configuration

## How to run

1. Install Node.js from https://nodejs.org/
2. Open a terminal in this project folder.
3. Run: `npm install`
4. Run: `npm run dev`
5. Open the local URL shown in the terminal, usually `http://localhost:5173/`

## Build for upload/hosting

Run `npm run build`. The production files will be created in the `dist/` folder.
